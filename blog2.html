地震情報APIって…？？

地震情報のAPIはアプリ開発やボット開発において、ほぼ必ず探されるものです。私もネット上を探し回りました。
そこで、ここを見ている皆さんのために、無料で使える地震情報のAPIを紹介していきます！

P2P地震情報API (無料)

こちらは知っている方も多いのではないでしょうか。私も現在利用しています。

特に WebSocket に対応している のが大きなメリットです。開発してみると WebSocket の便利さがよく分かります。また、JSON 形式で取得可能なのも嬉しいポイントです。

P2P地震情報APIの詳細はこちら

AXIS(無料　制限あり)

こちらの情報源は、地震情報のみならず気象情報の WebSocket 通信も可能です。ただし、GET 方式が使えない 点には注意が必要です。

低遅延で情報を取得できますが、現在は ベータテスト版 であり、以下の注意点があります。

現在、AXISはベータテスト版のサービスです。予期せぬサービス停止、配信メッセージの遅延が発生する可能性があります。

購読したメッセージは私的範囲内あるいは実験・研究目的でのみ利用可能です。再配信、転載、複製、改変、商用利用は禁止されています。

また、複数アカウントを作成して最大同時接続数を超える接続を行うことはご遠慮ください。（最大同時接続数の緩和を希望される場合はお問い合わせください）

AXISの詳細はこちら

Wolfx防災API(無料)

こちらは色んな情報に対応している意外と有名なものです。対応情報は、日本の緊急地震速報、地震情報、中国の地震情報などなど様々です。無料で使えるというところが素晴らしいのでぜひ使ってみてください。

最近まで中国語だったものの、日本語対応してました！！ありがたい。

Wolfx防災の詳細はこちら

DMDATA(有料)

こちらは、そうしさんが運営をしている気象庁が発表した情報のほぼすべてが入手できるAPIです。有料ですが、使う価値は絶対あると思います。（私は未使用です、、、）

このサービスはもっと「手軽に生の情報を素早く得たい」、という運営者の思いによって2020年04月に正式誕生しました。

WebSocketを通じて、気象庁（気象業務支援センター）の配信する"地震情報"や"津波情報"、 "気象警報"等を気象庁防災情報XMLで提供されている形式で配信する API を提供しています。

WebSocketを利用することで速報性のある緊急地震速報や、震度速報、津波警報、特別警報などを即座に配信できます。

このサービスを通して得た情報（気象庁発表の電文に限る）は無償・商用問わず二次利用が可能です（一部を除く。）。

気象業務支援センター（気象庁構内）の東京・大阪から常時リアルタイム配信を受けており、地理冗長の常時二系統運用を行っています。
